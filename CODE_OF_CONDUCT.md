# Code of Conduct

이 프로젝트는 서로를 존중하는 협업을 지향합니다.

## 기대 행동

- 다른 기여자의 배경, 경험, 관점을 존중합니다.
- 기술적 비판은 구체적인 근거와 개선 방향으로 표현합니다.
- 연구 결과를 과장하지 않고, 한계와 재현 조건을 명확히 설명합니다.

## 허용하지 않는 행동

- 모욕, 차별, 괴롭힘, 위협
- 의도적으로 오해를 부르는 성능 주장
- 보안상 위험한 코드나 우회성 임시 패치 제안

## 신고와 대응

문제가 발생하면 repository maintainer에게 issue 또는 private channel로 알려주십시오. Maintainer는 맥락을 검토하고 필요한 경우 comment 숨김, issue lock, 기여 제한 등의 조치를 취할 수 있습니다.
